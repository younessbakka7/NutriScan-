# NutriScan

## Description du projet

NutriScan est une application complète de suivi nutritionnel et de gestion des produits alimentaires, avec deux types d’utilisateurs principaux : l’Administrateur et le Client. L’application permet de gérer les produits, les utilisateurs et les préférences des utilisateurs de manière sécurisée et intuitive.

---

## Fonctionnalités pour l’Administrateur

- **Gérer les produits (Aliments)** : ajouter de nouveaux aliments, modifier des aliments existants, supprimer des aliments.
- **Gérer les utilisateurs** : afficher tous les utilisateurs qui se sont connectés à l’application, modifier leurs comptes si nécessaire.
- **Voir les favoris des utilisateurs** : consulter les aliments ajoutés aux favoris par chaque utilisateur.
- **Gérer son propre compte Administrateur** : modifier ses informations personnelles et ses paramètres.

---

## Fonctionnalités pour le Client

- **Authentification** : login pour accéder à son compte.
- **Scanner un code-barres** : obtenir des informations sur un aliment ou un produit spécifique (qualité, informations nutritionnelles…).
- **Voir la liste des aliments créés par l’Administrateur**.
- **Ajouter aux favoris** : mettre en favoris les aliments qu’il souhaite suivre.
- **Scanner des produits** : vérifier les informations de tout produit et l’ajouter à son suivi personnel.

---

## Objectif du projet

L’objectif de NutriScan est de permettre aux utilisateurs de suivre et gérer leur alimentation de manière efficace et personnalisée, tout en donnant aux administrateurs la possibilité de contrôler les produits, gérer les utilisateurs et analyser leurs préférences.

---

## Technologies utilisées

- **Frontend** : React Native / Expo
- **Backend** : Laravel (API)
- **Base de données** : MySQL
